# Weatherly - Weather Prediction App

# Description
A simple weather app written in python using customtkinter for the gui and visual-crossing weather api for the required weather data.

# Installation
Clone the repo into your device with :
```https://github.com/NorSomething/Weather-App.git```
Intall the required python modules with :
```pip install -r requirements.txt```

# Usage
Run the app with :
```python main.py```
